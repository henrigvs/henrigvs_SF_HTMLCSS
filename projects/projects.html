National register

This is a project developed for a course of algorithmic in 2022 to demonstrate the ability to develop recursive algorithms. The program, entirely specified with OpenJML (the code available of my GitHub Repository), takes in entry a file .txt including a list of names :

  • First line : n = number of tests in the file.
  • For each test, there is the number of names in the list and the list of names (current belgian names).

If for a name of the list, # of this name > n/2 => return this name. If not => return null.
2 types of algorithms are developed :

  • An iterative algorithm (Brute Force Algorithm).
  • A recursive algorithm (Divide & Conquer) empowered by Dynamic Programing.
